--- layout: page title: ES6 Quick Check description: Check if your browser supports ES6 features that OctoPrint uses --- This page allows you to do a quick check whether your browser supports the ES6 features that OctoPrint uses.
Feature Required since Supported?
arrow functions 1.9.0
async/await 1.9.0
generator 1.9.0
let 1.9.0
If any of the tests above say "No", you will need to use a different browser or update your current browser to a newer version before you can use OctoPrint.
